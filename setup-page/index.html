Perplexity Research for Claude Desktop

Intelligent web research with automatic knowledge logging

Smart Model Selection
Auto-picks the right Perplexity model for your query
Research History
Saves all research to ~/Documents/Perplexity Research/
Multi-Model Synthesis
Combines multiple models for comprehensive research
Built-in Citations
Every response includes sources and references

Prerequisite: You need Node.js installed. Check with: node --version in Terminal.

1
Get a Perplexity API Key
  1. Go to perplexity.ai/settings/api
  2. Sign in or create an account
  3. Click "Generate API Key"
  4. Copy the key (starts with pplx-)

Note: Perplexity API has usage costs. Check their pricing page for details. Typical research queries cost a few cents each.

2
Enter Your API Key
Your Claude Desktop Config:
Copied to clipboard!
3
Add to Claude Desktop
  1. Open Finder
  2. Press Cmd + Shift + G
  3. Paste: ~/Library/Application Support/Claude/
  4. Open or create claude_desktop_config.json
  5. Paste the config from Step 2
  6. Save the file
4
Restart Claude Desktop

Quit Claude Desktop completely (Cmd + Q) and reopen it.

You should now see the Perplexity tools available when you ask Claude to research something!

5
Try It Out

Ask Claude something like:

  • "Research the current state of AI in healthcare"
  • "What are the best practices for React performance in 2025?"
  • "Do a deep research on renewable energy trends"

Your research will be saved to ~/Documents/Perplexity Research/
